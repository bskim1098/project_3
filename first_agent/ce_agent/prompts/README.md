# CE agent prompts

- `claim_extraction_prompt.md`: 현재 CE agent의 LangChain LLM 호출에서 실제로 사용하는 프롬프트입니다. 기사 제목과 본문에서 핵심 주장을 structured output으로 요약합니다.
- `claim_evidence_prompt.md`: 전체 CE agent의 역할, 출력 계약, 안전 원칙을 설명하는 정책 문서입니다. 현재 LangChain 호출에 직접 연결되지는 않습니다.
- 나머지 프롬프트 파일은 단계별 역할을 기록한 설계 자료이며 현재 규칙 기반 노드를 대체하지 않습니다.
