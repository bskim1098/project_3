# Claim Evidence Agent Plan

담당자: 준영님

책임 비중: 전체 멀티에이전트 업무의 **60%**

주요 책임:

1. 표·차트의 수치, 축, 단위, 항목, 변화량 추출
2. 기사 제목과 본문의 핵심 주장 추출
3. 폭증, 급락, 압도, 역대급, 원인 등 강한 표현 탐지
4. 기사 주장과 뉴스 내부 시각자료 비교
5. 보수적인 1차 판정 생성
6. GraphRAG를 이용한 관계 검색과 출처 보조 확인(도입 예정)

이 에이전트는 `ce_` 결과만 작성하며, 결과는 3번 `vc_` 최종판정 검토 에이전트로 전달한다.
세부 노드·프롬프트·GraphRAG 로직은 준영님이 구현할 예정이다.
